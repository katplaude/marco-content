
This repository contains a Strapi backend (`content-strapi`) and a React frontend (`content-frontend`)

 Start Strapi Backend:
    cd content-strapi
    npm install
    npm run develop 
    (Runs on `http://localhost:1337`)

 Start React Frontend:
    cd content-frontend
    npm install 
    npm start 
    (Runs on `http://localhost:3000`)
